<static> prompt(message, resultCallback, title, buttonLabels, defaultText)

Open a native prompt dialog, with a customizable title and button text.
The following results are returned to the result callback:
buttonIndex Index number of the button selected.
input1 The text entered in the prompt dialog box.

Parameters:
Name Type Description
message String

Dialog message to display (default: "Prompt message")
resultCallback function

The callback that is called when user clicks on a button.
title String

Title of the dialog (default: "Prompt")
buttonLabels Array

Array of strings for the button labels (default: ["OK","Cancel"])
defaultText String

Textbox input value (default: empty string)
