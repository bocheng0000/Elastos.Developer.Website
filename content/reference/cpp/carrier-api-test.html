Data types¶

Bootstrap¶

struct BootstrapNode¶

Bootstrap defines a couple of perperities to provide for Carrier nodes to connect with.

The bootstrap nodes help Carrier nodes be connected to the others with more higher possibilities.

Public Members

const char *ipv4¶

The ip address supported with ipv4 protocol.

const char *ipv6¶

The ip address supported with ipv6 protocol.

const char *port¶

The ip port.

The default value is 33445.

const char *public_key¶

The unique public key to provide for Carrier nodes, terminated by null-string.

The length of public key is about 45 bytes.

ElaOptions¶

struct ElaOptions¶

ElaOptions defines several settings that control the way the Carrier node connects to others.

Remark
Default values are not defined for persistent_location of Carrier- Options, so application should be set persistent_location clearly. If the ElaOptions structure is defined as a static variable, initialization (in compliant compilers) sets all values to 0 (NULL for pointers).

Public Members

const char *persistent_location¶

The application defined persistent data location.

The location must be set.

bool udp_enabled¶

The option to decide to use udp transport or not.

Setting this option to false will force Carrier node to use TCP only, which will potentially slow down the message to run through.

size_t bootstraps_size¶

The total number of DHT bootstrap nodes to connect.

There must have at least one bootstrap node for the very first time to create carrier instance.

BootstrapNode *bootstraps¶

The array of DHT bootstrap nodes.

size_t hive_bootstraps_size¶

The total number of Hive bootstrap nodes to connect.

There must have at least one bootstrap node for the very first time to create carrier instance.

HiveBootstrapNode *hive_bootstraps¶

The array of Hive bootstrap nodes.

ElaConnectionStatus¶

enum ElaConnectionStatus¶

Carrier node connection status to Carrier network.

Values:

ElaConnectionStatus_Connected¶

Carrier node connected to Carrier network.

Indicate the Carrier node is online.

ElaConnectionStatus_Disconnected¶

There is no connection to Carrier network.

Indicate the Carrier node is offline.

ElaPresenceStatus¶

enum ElaPresenceStatus¶

Carrier node presence status to Carrier network.

Values:

ElaPresenceStatus_None¶

Carrier node is online and available.

ElaPresenceStatus_Away¶

Carrier node is being away.

Carrier node can set this value with an user defined inactivity time.

ElaPresenceStatus_Busy¶

Carrier node is being busy.

Carrier node can set this value to tell friends that it can not currently wish to commincate.

ela_run¶

int ela_run(ElaCarrier *carrier, int interval)¶

Attempts to connect the node to Carrier network.

If the node successfully connects to Carrier network, then it starts the node’s main event loop. The connect options was specified by previously create options.

See
ela_new().
Return
0 if the client successfully connected to Carrier network and start the event loop. Otherwise, return -1, and a specific error code can be retrieved by calling ela_get_error().
Parameters
  • carrier: [in] A handle identifying the Carrier node instance.
  • interval: [in] Internal loop interval, in milliseconds.

ela_kill¶

void ela_kill(ElaCarrier *carrier)¶

Disconnect from Carrier network, and destroy all associated resources with the Carrier node instance.

After calling the function, the Carrier pointer becomes invalid. No other functions can be called.

Parameters
  • carrier: [in] A handle identifying the Carrier node instance to kill.
